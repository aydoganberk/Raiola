# Raiola Codex Optimizer

This plugin packages Raiola as a native Codex operating layer.

Included surfaces:

- native `.codex/config.toml` generation
- native hooks under `.codex/hooks.json`
- project subagents under `.codex/agents/*.toml`
- layered `AGENTS.md` guidance
- installable skill pack under `skills/*`
- Codex cockpit launch-kit support with runnable launcher scripts and continuity files
- native telemetry summaries driven by hook-captured flight-recorder events
- first-party GitHub review prompt under `.github/codex/prompts/review.md`
